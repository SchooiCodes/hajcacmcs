## Δημιουργία (Hosting)

### Απαιτήσεις
Οι μόνες απαιτήσεις είναι:
1. Ένας Minecraft launcher  
2. Ένας λογαριασμός Aternos

### Δημιουργία λογαριασμού Aternos
Για να δημιουργήσεις λογαριασμό, πήγαινε στο https://aternos.org/signup/.  
Εκεί, διάλεξε ένα όνομα χρήστη, ενεργοποίησε τα checkbox και πάτησε "**Next**".  
Έπειτα, διάλεξε κωδικό πρόσβασης και προαιρετικά πρόσθεσε email για ανάκτηση.  
Εναλλακτικά, μπορείς να συνδεθείς με Google.

### Δημιουργία του server
Αφού συνδεθείς στο Aternos, πήγαινε στο https://aternos.org/servers/.  
Εκεί, πάτησε "**Create a server**".  
Έπειτα, άλλαξε το όνομα και το MOTD αν θέλεις.  
**ΜΗΝ αλλάξεις την πλατφόρμα του server. Άφησέ τη στο “Java Edition”.**  
Πάτησε "**Create**".  
Αν θέλεις να κάνεις backup τον server, ακολούθησε τα βήματα. Αλλιώς πάτησε skip.  
Θα βρεθείς στη σελίδα https://aternos.org/server/.

Διάλεξε "**Paper / Bukkit”**" (https://aternos.org/software/papermc/).  
Διάλεξε την τελευταία έκδοση ή την έκδοση του κόσμου σου αν έχεις ήδη server.  
Πάτησε "**Install**".

https://aternos.org/addons/a/modrinthbukkit/P1OZGk5p (ViaVersion)

- https://aternos.org/addons/a/modrinthbukkit/NpvuJQoq (ViaBackwards)

Μετά από αυτό, ο server σου μπορεί να δεχτεί παίκτες από **οποιαδήποτε έκδοση Minecraft**.

## Σύνδεση (Joining)

Για να συνδεθείς, ο server πρέπει να είναι ανοιχτός.  
Για να τον ξεκινήσεις, πήγαινε στο https://aternos.org/server/ και πάτησε "**Start**".

Πριν συνεχίσουμε, χρειάζεσαι:
1. Το **Server IP**
2. Το **Server Port**

Για να τα βρεις, πάτησε "**Server**" στο πλαϊνό μενού (https://aternos.org/server/).  
Έπειτα πάτησε "**Connect**" κάτω από το IP του server.  
Σημείωσε τις πληροφορίες.

---

### Java Edition
Η πλήρης διεύθυνση του server είναι:
**[Server IP]:[Server Port]**  
π.χ. `example.aternos.me:12345`

Στο Multiplayer:
- Πάτησε "**Add Server**"
- Δώσε όνομα στον server
- Στο Address βάλε την πλήρη διεύθυνση
- Πάτησε "**Done**"
- Κάνε διπλό κλικ για να μπεις

---

### Bedrock Edition
Στο κύριο μενού πάτησε "**Play**".  
Πήγαινε στην καρτέλα "**Servers**".  
Πάτησε "**Add Server**".

Δεν βλέπεις το κουμπί; Δες το "**Σύνδεση χωρίς το κουμπί Add Server**".

Συμπλήρωσε:
- **Server Name**: ό,τι θέλεις  
- **Server Address**: example.aternos.me  
- **Server Port**: 12345  

Πάτησε "**Save**" και μπες στον server.
